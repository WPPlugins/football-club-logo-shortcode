A plugin which lets you add Football Club's logo in posts and pages with shortcode.

== Description ==

Football Club Logo Shortcodes will let you display logo of English Premier League football club's logo anywhere in your wordpress website (posts or pages) using shortcodes.

== Installation == 

1. Upload the entire football-club-logo-shortcode folder to the /wp-content/plugins/ directory.

OR

Upload football-club-logo-shortcode.zip using your website’s plugin uploader.

2. Activate the plugin using the ‘Plugins’ menu in your WordPress website dashboard.
